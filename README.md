O portfólio reúne informações profissionais, habilidades, projetos desenvolvidos e formas de contato. A interface foi construída com foco em:

- Design moderno e profissional
- Responsividade para desktop, tablet e celular
- Navegação simples e intuitiva
- Acessibilidade e HTML semântico
- Boa organização do código
- Apresentação clara dos projetos e tecnologias

## Projetos apresentados

### Dev Store

E-commerce voltado ao público desenvolvedor, criado com foco em design moderno, componentização e navegação fluida.

**Tecnologias:** React, JavaScript e CSS.

### Alura Space

Galeria de imagens desenvolvida com persistência de dados, templates, views, modelos e painel administrativo.

**Tecnologias:** Python, Django e banco de dados.

### Lista de Tarefas

Aplicação para gerenciamento de tarefas com operações de criação, leitura, atualização e exclusão de dados.

**Tecnologias:** Django, CRUD e Class-Based Views.

### Alura Plus

Landing page responsiva para divulgação de uma plataforma de streaming.

**Tecnologias:** HTML5 e CSS3.

### Alura Books

Plataforma de livros com filtragem e manipulação dinâmica de conteúdo.

**Tecnologias:** JavaScript, DOM e CSS.

### Apple Watch

Página interativa de produto com alteração dinâmica de cores, pulseiras e visualizações.

**Tecnologias:** JavaScript, manipulação de estado e interatividade.

### Cardápio Online

Cardápio digital responsivo com apresentação dinâmica de produtos, categorias e preços.

**Tecnologias:** React, JavaScript e componentes.

## Como executar

1. Faça o download ou clone o repositório:

```bash
git clone https://github.com/Gu1Andrade/SEU-REPOSITORIO.git
```

2. Acesse a pasta do projeto:

```bash
cd SEU-REPOSITORIO
```

3. Abra o arquivo `index.html` no navegador.

Para uma melhor experiência durante o desenvolvimento, também é possível utilizar a extensão **Live Server** no Visual Studio Code.
